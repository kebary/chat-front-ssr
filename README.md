# chat front server side rendering

## Versions
```
$ node -v
v8.9.1

$ npm -v
5.5.1
```

## Install
```
anyenv
https://github.com/riywo/anyenv
or
ndenv
https://github.com/riywo/ndenv
```
```
$ npm install
```

## Server
```
$ npm run start
```

## build
```
$ npm run build
```
